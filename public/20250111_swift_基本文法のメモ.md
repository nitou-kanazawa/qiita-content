#### 変数と定数

変数を`var`，定数を`let`で宣言する

```
// 変数
var mutable = "Apple"
mutable = "Lemon"

// 定数
let immutable = "Apple"
immutable = "Lemon"   // Error - Cannot assign to value: 'immutable' is a 'let' constant
```
